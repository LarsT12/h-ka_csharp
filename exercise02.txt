Aufg. 1
  Erstelle eine Klasse namens "Block" mit
  einer Ausgabe zur Begrüßung.
  
Aufg. 2
  Deklariere eine Variable j vom Typ int innerhalb der Main-Methode und initialisiere sie mit einem Wert.
  Deklariere eine weitere Variable k vom Typ int innerhalb eines Blocks {}, den Sie in der Main-Methode erstellen.
  Geben Sie in dem Block eine Berechnung (z.B. Multiplikation) der beiden Variablen j und k auf der Console aus.
  
Aufg. 3
  Versuchen Sie dieselbe Berechnung außerhalb des Blocks auf der Console auszugeben und interpretieren Sie die Fehlermeldung, 
  die beim Kompilieren erscheint.
  
Aufg. 4
  Erklären Sie, wie sich die Sichtbarkeit von Variablen von innen nach außen verhält. 
  Erklären Sie ebenfalls die Sichtbarkeit von Variablen von außen nach innen - 
  betrachten Sie dazu die in Aufg. 2 durchgeführte Berechnung.

Aufg. 5
  Deklarieren Sie erneut die Variable j als static int außerhalb der Main-Methode, 
  jedoch innerhalb der Klasse, und weise ihr einen Wert zu.
  Erstellen Sie in der Main-Methode eine Verzweigung (Selektion), 
  die in Abhängigkeit von j einen Block ausführt oder nicht. Analysieren Sie, welche Variable verwendet wird, 
  die außerhalb der Main-Methode oder die innere?
